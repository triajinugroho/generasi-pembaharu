# Panduan Berkontribusi

Terima kasih atas niat Anda untuk berkontribusi dalam proyek **Generasi Pembaharu**. Setiap kontribusi adalah amal jariyah yang insya Allah akan terus mengalir manfaatnya.

## Model Kolaborasi: Kurated Community

Repo ini bukan open-source bebas. Kami menerapkan model **kurated community**, artinya:

- Siapa pun boleh mengajukan kontribusi
- Setiap kontribusi melalui **editorial review** sebelum diterima
- Kualitas, kedalaman, dan keseimbangan manhaj adalah standar utama

## Jenis Kontribusi yang Diterima

### 1. Perbaikan Konten
- Koreksi ayat, hadits, atau referensi yang keliru
- Perbaikan tata bahasa dan ejaan
- Penambahan referensi yang relevan

### 2. Penambahan Materi
- Esai baru yang sesuai dengan visi repo
- Pendalaman topik yang sudah ada
- Studi kasus atau contoh kontekstual Indonesia

### 3. Terjemahan & Adaptasi
- Terjemahan materi ke bahasa daerah
- Adaptasi kurikulum untuk konteks lokal

### 4. Teknis
- Perbaikan link dan navigasi
- Peningkatan format dan keterbacaan

## Prinsip Penulisan

Sebelum menulis, pahami prinsip-prinsip berikut:

1. **Berbasis dalil** — Setiap klaim harus bisa dirujuk ke Al-Quran, hadits shahih, atau karya ulama muktabar
2. **Lintas manhaj** — Jangan menyerang organisasi atau manhaj tertentu. Ambil hikmah, tinggalkan khilafiyah yang memecah
3. **Bahasa hidup** — Tulis dengan bahasa yang kuat dan menggugah, bukan akademik kering atau ceramah populer
4. **Kontekstual Indonesia** — Materi harus membumi di realitas pemuda Muslim Indonesia
5. **Actionable** — Akhiri dengan refleksi atau langkah aksi nyata

## Cara Berkontribusi

### Langkah 1: Fork & Clone
```bash
git fork https://github.com/triajinugroho/generasi-pembaharu
git clone <your-fork-url>
```

### Langkah 2: Buat Branch
```bash
git checkout -b kontribusi/topik-anda
```

### Langkah 3: Tulis Kontribusi
- Ikuti struktur file yang sudah ada
- Sertakan penanda level jika menambah materi baru: `> **Level**: Foundational | Intermediate | Advanced`
- Gunakan format Markdown yang konsisten

### Langkah 4: Submit Pull Request
- Judul PR yang jelas dan deskriptif
- Jelaskan apa yang Anda tambahkan/ubah dan mengapa
- Sertakan referensi jika menambah konten baru

## Proses Review

1. **Review Teknis** — Format, link, konsistensi struktur
2. **Review Konten** — Keakuratan dalil, kedalaman pembahasan
3. **Review Manhaj** — Keseimbangan, inklusivitas, tidak partisan
4. **Review Bahasa** — Keterbacaan, kekuatan narasi

Proses review bisa memakan waktu. Bersabarlah — ini demi menjaga kualitas yang menjadi amanah kita bersama.

## Kode Etik

- Niatkan kontribusi sebagai ibadah
- Hormati perbedaan manhaj dan organisasi
- Kritik ide, bukan pribadi atau kelompok
- Jaga adab dalam diskusi dan review
- Utamakan persatuan di atas perbedaan

## Kontak

Untuk diskusi lebih lanjut, silakan buka Issue di repo ini atau hubungi pengelola melalui channel yang tersedia.
