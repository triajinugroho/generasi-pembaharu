# Cara Menggunakan Repositori Ini

> *"Barangsiapa menempuh jalan untuk mencari ilmu, Allah akan mudahkan baginya jalan menuju surga."* — HR. Muslim

Selamat datang di **Generasi Pembaharu**. Repositori ini berisi buku, kurikulum, esai, dan referensi untuk gerakan dakwah pemuda Islam Indonesia. Kamu tidak perlu membaca semuanya sekaligus. Temukan jalurmu, mulai dari satu langkah, dan biarkan perjalanan ini membentukmu.

## Jalur 1: Aktivis Baru

**Baru Bergabung di LDK/Rohis**

*Tidak perlu baca semuanya. Mulai pelan-pelan. Yang penting: konsisten.*

### Quick Start (3 Bacaan Pertama)

1. **[Manifesto Generasi Pembaharu](./00-mukadimah/01-manifesto-generasi-pembaharu.md)** — Kenali visi besar yang kamu masuki
2. **[Peta Krisis Pemuda Islam](./00-mukadimah/02-peta-krisis-pemuda-islam-indonesia.md)** — Pahami mengapa gerakanmu penting
3. **[Tauhid sebagai Pembebasan](./01-fondasi-aqidah/01-tauhid-sebagai-pembebasan.md)** — Fondasi pertama: iman yang membebaskan

### Peta 30 Hari Pertama

| Minggu | Bacaan | Aksi |
|--------|--------|------|
| 1 | Mukadimah (3 file) | Tulis: "Mengapa aku di sini?" di jurnal |
| 2 | Fondasi Aqidah — Tauhid sebagai Pembebasan | Diskusikan dengan murabbi/teman |
| 3 | Sirah — Blueprint Nabawi | Catat 3 pelajaran dari sirah Nabi |
| 4 | Pembangunan Diri — Muraqabah dan Muhasabah | Mulai kebiasaan muhasabah harian |

### Lanjutkan ke Kurikulum

Setelah 30 hari pertama, masuk ke **[Kurikulum Level 1 — Tamhidi (Persiapan)](./kurikulum/level-1-tamhidi/)**. Ini adalah program terstruktur 6 sesi yang dirancang khusus untuk pemula.

### Tips

- Jangan takut bertanya ke murabbi-mu
- Catat setiap pertanyaan yang muncul — itu tandanya kamu bertumbuh
- Ikuti satu halaqah secara rutin, jangan coba pelajari semuanya sendirian
- Baca [Surat untuk Aktivis Kampus](./esai/surat-untuk-aktivis-kampus.md) jika kamu merasa kewalahan

## Jalur 2: Murabbi / Fasilitator Halaqah

**Menjalankan Halaqah dengan Materi yang Terstruktur**

*Semua yang kamu butuhkan untuk menjalankan halaqah ada di sini — dari materi hingga evaluasi.*

### Quick Start

1. **[Panduan Fasilitator](./lampiran/panduan-fasilitator.md)** — Prinsip, teknik, dan template sesi 120 menit
2. **[Kurikulum README](./kurikulum/)** — Filosofi dan cara menggunakan kurikulum 3 level
3. **[Template Evaluasi](./lampiran/template-evaluasi.md)** — Rubrik penilaian per level

### Cara Memilih Level untuk Peserta

| Profil Peserta | Level yang Tepat |
|----------------|------------------|
| Baru bergabung, belum punya fondasi keislaman yang kuat | **[Tamhidi (Persiapan)](./kurikulum/level-1-tamhidi/)** — 6 sesi fondasi |
| Sudah aktif 1-2 tahun, mulai memegang amanah organisasi | **[Muayyid (Pendukung)](./kurikulum/level-2-muayyid/)** — 6 sesi pendalaman |
| Kader inti, calon pemimpin, siap menjadi murabbi | **[Muntasib (Anggota Penuh)](./kurikulum/level-3-muntasib/)** — 6 sesi kepemimpinan |

### Peta Materi per Level

**Tamhidi:** Jati Diri → Aqidah Dasar → Teladan Rasul → Adab Da'i → Dakwah Fardiyah → Evaluasi & Komitmen

**Muayyid:** Pendalaman Tauhid → Sirah Strategis → Kepemimpinan → Dakwah Digital → Analisis Gerakan → Proyek Dakwah

**Muntasib:** Falsafah Pergerakan → Strategi Makro → Membangun Institusi → Kolaborasi Lintas Gerakan → Tantangan Zaman → Visi & Aksi Pembaharu

### Materi Pengayaan dari Buku Utama

Setiap sesi kurikulum memiliki cross-reference ke bab buku utama. Gunakan bab-bab ini sebagai bacaan pengayaan untuk memperdalam diskusi:

- **Bab 1-2** untuk Level Tamhidi (fondasi aqidah + sirah)
- **Bab 3-4** untuk Level Muayyid (pembangunan diri + strategi dakwah)
- **Bab 5-6** untuk Level Muntasib (titik temu gerakan + tantangan kontemporer)

### Referensi Bacaan untuk Peserta

Gunakan **[Peta Bacaan Bertahap](./lampiran/peta-bacaan-bertahap.md)** untuk merekomendasikan kitab/buku sesuai level peserta.

*Bangun program yang berkelanjutan — bukan hanya kegiatan sesaat.*

### Quick Start

| Bulan | Program | Sumber Materi |
|-------|---------|---------------|

| 3-4 | Pembinaan Fondasi | Kurikulum Tamhidi Sesi 3-6 |

### Integrasi Kurikulum ke Kalender Organisasi

- **Semester Ganjil:** Fokus Level Tamhidi untuk anggota baru + Level Muayyid untuk pengurus

- **Liburan Semester:** Daurah (pelatihan intensif) menggunakan esai dan bab terpilih

## Jalur 4: Pembaca Mandiri

**Belajar Sendiri, di Mana Saja, Kapan Saja**

*Belajar sendiri juga mulia. Yang penting: punya peta jalan dan disiplin.*

### Quick Start

Mulai dari **[Mukadimah](./00-mukadimah/)** dan ikuti urutan bab secara berurutan:

```
Mukadimah → Fondasi Aqidah → Sirah & Sejarah → Pembangunan Diri
    → Strategi Dakwah → Titik Temu Gerakan → Tantangan Kontemporer → Penutup
```

### Panduan Membaca

| Kecepatan | Jadwal | Estimasi Selesai |
|-----------|--------|------------------|
| Santai | 1 file per minggu | ~8 bulan |
| Moderat | 2-3 file per minggu | ~3-4 bulan |
| Intensif | 1 file per hari | ~6 minggu |

### Tips untuk Pembaca Mandiri

1. **Siapkan jurnal refleksi** — Tulis pertanyaan, insight, dan rencana amal setelah setiap bacaan
2. **Gunakan [Peta Bacaan Bertahap](./lampiran/peta-bacaan-bertahap.md)** untuk rekomendasi kitab/buku pendukung
3. **Cari teman diskusi** — Buku yang dibaca sendirian hanya separuh manfaatnya
4. **Perhatikan penanda level** pada setiap file: [F] = Foundational, [I] = Intermediate, [A] = Advanced

### Bacaan Tematis (Jika Tidak Ingin Urut)

| Minat | File yang Relevan |
|-------|-------------------|
| Spiritualitas | Bab 3 (Pembangunan Diri) + Esai "Spiritualitas di Era Digital" |
| Kepemimpinan | Bab 3 file 4 (Kepemimpinan Profetik) + Bab 4 (Strategi) |
| Sejarah Islam | Bab 2 (Sirah & Sejarah) lengkap |
| Isu Kontemporer | Bab 6 (Tantangan Kontemporer) + Esai terkait |
| Pergerakan | Bab 5 (Titik Temu) + Esai "Dari Halaqah ke Peradaban" |

### Catatan Metodologis

Repositori ini menggunakan pendekatan **lintas manhaj** — mengambil hikmah dari berbagai aliran tanpa menyerang satu pun. Ia bukan representasi satu organisasi, melainkan sintesis dari berbagai tradisi dakwah kampus di Indonesia: dari yang menekankan tarbiyah, tasfiyah, aksi sosial, maupun spiritualitas.

## FAQ

**"Harus baca dari awal?"**
Tidak. Pilih jalur sesuai profilmu di atas. Setiap jalur punya titik masuk yang berbeda.

**"Bisa dipakai lintas organisasi?"**
Ya. Repositori ini **lintas manhaj** — bukan milik satu organisasi. Siapa pun bisa menggunakannya untuk kepentingan dakwah.

**"Materi ini terlalu banyak, dari mana mulainya?"**
Mulai dari **3 file Quick Start** di jalurmu. Jangan lihat keseluruhan — fokus pada langkah pertama saja.

**"Bagaimana jika saya ingin berkontribusi?"**
Baca [CONTRIBUTING.md](./CONTRIBUTING.md) untuk panduan kontribusi. Repositori ini bersifat kurated community — kontribusi terbuka melalui proses editorial review.

**"Apakah ada versi cetak?"**
Belum tersedia. Untuk saat ini, konten ini dirancang untuk dibaca secara digital.
