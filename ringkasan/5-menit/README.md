# Ringkasan 5 Menit per Bab

~700 kata per bab. Cukup untuk:

- Broadcast grup WhatsApp
- Newsletter
- Artikel pembuka majalah kampus
- Bacaan di perjalanan komuter (5-10 menit di KRL/TransJ)

## Status

_Direktori ini belum diisi._ Kontributor silakan membuat draf berbasis template di `TEMPLATE.md` (akan ditambahkan oleh kontributor pertama).

## Ide Kerangka (700 kata)

1. **Konteks & pertanyaan inti** (~100 kata)
2. **Inti pembahasan** — 3 sub-poin × ~150 kata = 450 kata
3. **Satu dalil kunci + syarahnya** (~80 kata)
4. **Aksi & refleksi** (~50 kata)
5. **Link ke bab lengkap + sesi halaqah terkait** (~20 kata)

## Cara Berkontribusi

1. Baca bab asli secara utuh.
2. Tulis draf 700 kata — simpan sebagai `ch<N>.md` (N = 0-7).
3. Minta review dari murabbi atau kontributor lain sebelum PR.
4. Pastikan tidak mengerdilkan nuansa bab asli.
