# Jalur Baca per Persona

> *"Dan tidaklah Kami mengutus seorang rasul pun melainkan dengan bahasa kaumnya."*
> — QS. Ibrahim [14]: 4

Repo ini cukup besar. Membacanya dari `00-mukadimah` sampai `07-penutup` berurutan memakan ~8 jam bacaan lambat. Sebagian besar pembaca **tidak butuh semuanya** — yang dibutuhkan adalah **jalur yang sesuai dengan peran dan pertanyaan mereka saat ini**.

Lima jalur di bawah ini adalah peta yang telah dikurasi. Setiap jalur:

- berbentuk **checklist** (pakai `- [ ]`) supaya Anda bisa fork repo, tandai sendiri progresnya;
- memberi **estimasi waktu** realistis;
- diakhiri dengan **aksi konkret** — bukan sekadar "selesai baca".

## Cara Memakai Jalur (Fork-and-Track)

1. **Fork repo ini** ke akun GitHub Anda.
2. Buka file jalur Anda di fork.
3. Centang `[ ]` → `[x]` saat menyelesaikan setiap item (GitHub menyimpannya sebagai commit personal Anda).
4. Tulis catatan refleksi Anda di bawah tiap item (commit langsung di browser).
5. Selesai satu jalur → pindah ke jalur berikutnya atau dalami bab tematik.

Ini memberi Anda **jurnal belajar pribadi** sekaligus **jejak kontribusi**: kalau ada item yang Anda lengkapi dengan insight baru, buka PR ke repo utama — kami senang menerima pengalaman lapangan.

---

## Alternatif Non-GitHub

Kalau tidak mau fork:

- **Download jalur Anda sebagai PDF**: klik ⋯ di GitHub > Print > Save as PDF.
- **Salin ke Notion/Obsidian**: paste isi file, checkbox tetap berfungsi.
- **Cetak**: 1-3 halaman, tempel di dinding kamar.
