# Bab 5: Titik Temu Gerakan

> *"Dan berpeganglah kamu semuanya kepada tali (agama) Allah, dan janganlah kamu bercerai-berai."*
> — QS. Ali Imran [3]: 103

## Mengapa Bab Ini Ada

Inilah bab yang mungkin paling dibutuhkan — dan paling sulit. Gerakan dakwah di Indonesia terpecah dalam sekat-sekat organisasi yang saling curiga. Bab ini hadir bukan untuk menyamaratakan semua, tapi untuk **mencari titik temu dan membangun kolaborasi** di atas keragaman yang ada.
