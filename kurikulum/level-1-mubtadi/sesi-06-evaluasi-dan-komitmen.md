# Sesi 6: Evaluasi dan Komitmen

> **Level**: Mubtadi (Pemula) | **Durasi**: 120 menit

## Tujuan Pembelajaran
1. Peserta mereview dan mengkonsolidasikan semua pelajaran dari Level 1
2. Peserta mengevaluasi pertumbuhan pribadinya selama program
3. Peserta membuat komitmen tertulis untuk langkah selanjutnya

## Pembukaan (10 menit)
- Tilawah bersama: QS. Al-Ashr
- Refleksi: "Apa yang paling berubah dalam dirimu sejak sesi pertama?"

## Review Materi (30 menit)

### Kuis Interaktif
- Pertanyaan dari setiap sesi sebelumnya
- Format: tim vs. tim untuk membangun semangat
- Topik: jati diri, aqidah, sirah, akhlak, dakwah fardiyah

### Sharing Circle
- Setiap peserta berbagi: momen paling berdampak selama program
- Apa yang sudah berubah? Apa yang masih sulit?

## Evaluasi Diri (25 menit)

### Formulir Evaluasi
Peserta mengisi evaluasi diri untuk setiap aspek:

| Aspek | Sebelum (1-10) | Sesudah (1-10) |
|-------|---------------|----------------|
| Pemahaman jati diri sebagai Muslim | | |
| Kekuatan aqidah dan tauhid | | |
| Keteladanan terhadap Rasulullah | | |
| Kualitas akhlak dan adab | | |
| Keberanian berdakwah | | |
| Konsistensi ibadah harian | | |

### Refleksi Tertulis
- Apa 3 pelajaran terpenting yang kau dapat?
- Apa 1 perubahan terbesar yang sudah terjadi?
- Apa tantangan terbesar yang masih kau hadapi?

## Komitmen dan Ikrar (25 menit)

### Menulis Surat untuk Diri Sendiri
- Tulis surat untuk dirimu 6 bulan ke depan
- Isi: komitmen, target, dan harapan
- Surat akan disimpan dan dibuka 6 bulan kemudian

### Ikrar Bersama
Peserta berdiri dan berikrar bersama:

> *"Dengan menyebut nama Allah, aku berkomitmen untuk:
> terus belajar dan mendalami Islam,
> menjaga ibadah dan akhlakku,
> berdakwah dengan hikmah dan kasih sayang,
> dan menjadi pembaharu di lingkunganku.
> Allah saksi atas komitmenku ini."*

## Penutup (30 menit)
- Penyerahan sertifikat (jika ada)
- Penjelasan tentang Level 2: Mutawassith
- Doa bersama dan makan bersama
- Foto bersama

## Tindak Lanjut
- Peserta dimasukkan ke grup alumni Level 1
- Jadwal follow-up bulanan
- Pendaftaran Level 2 dibuka
- Tugas berkelanjutan: jurnal harian selama 30 hari
