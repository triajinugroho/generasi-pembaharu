## Cara Melengkapi

Indeks ini belum menyeluruh. Kontribusi:

1. Ambil satu istilah dari [`referensi/glosarium.md`](../referensi/glosarium.md).
2. Cari istilah itu di repo (pakai GitHub search atau `grep` di fork lokal).
3. Daftarkan 2-5 file paling relevan.
4. Kalau istilah belum di glosarium, pertimbangkan apakah wajar ditambahkan di sana juga.
5. Buka PR.
